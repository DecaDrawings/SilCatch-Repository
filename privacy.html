Política de Privacidade

Última atualização: Julho de 2025

A extensão SilCatch foi desenvolvida para uso interno em ambientes hospitalares com o objetivo exclusivo de facilitar a visualização e extração de exames laboratoriais diretamente do navegador.

Coleta de Dados

Esta extensão não coleta, armazena nem transmite qualquer dado pessoal, médico ou sensível. Toda a análise e extração de dados ocorrem exclusivamente no navegador local do usuário.

Permissões

As permissões solicitadas pela extensão são necessárias para:

  • Acessar o conteúdo da aba ativa e extrair exames laboratoriais visualizados na tela.
  • Executar scripts de leitura de dados localmente (sem envio para servidores).

Essas permissões são usadas apenas durante o uso da extensão e nunca para coleta remota ou rastreamento.

Armazenamento de Dados

Nenhuma informação é armazenada em servidores. Os dados são processados localmente e, se armazenados, permanecem apenas no navegador do usuário (por exemplo, em cache ou localStorage).

Compartilhamento de Dados

Não há compartilhamento de dados com terceiros, prestadores de serviços ou plataformas externas.

Segurança

A extensão foi desenvolvida com foco em segurança e privacidade. Nenhuma funcionalidade envia dados automaticamente para fora do computador do usuário.
